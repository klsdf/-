# 标题

最多支持六级标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```



# 列表

- 无序列表

```markdown
- 无
- 序
- 列
- 表
```

或者

```markdown
+ 无
+ 序
+ 列
+ 表
```

或者

```markdown
* 无
* 序
* 列
* 表
```



- 有序列表

```markdown
1. 有
2. 序
3. 列
4. 表
```



# 文字语法

- 缩小

```markdown
~小号字体~
```

- 斜体

```markdown
*斜体*
```

- 粗体

```markdown
**粗体**
```

- 斜体加粗

```markdown
***斜体加粗***
```

- 删除线

```markdown
~~删除线~~
```

# 表格

```markdown
| 表头 | 表头 | 表头 |
| ---- | :--: | ---: |
| 内容 | 内容 | 内容 |
| 内容 | 内容 | 内容 |
```

# 代码块

- 单行代码

```markdown
`单行代码`
```

- 多行代码

~~~markdown

	多行代码
```
~~~

# 公式


```

```
